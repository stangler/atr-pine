# ATR Volatility Filter — PineScript v6

株デイトレ用 ATRフィルター。エントリー可否を自動判定。

---

## トレードルール（変更不可）

- TP: **+1.20%** / SL: **-0.60%** / RRR: **2.0**
- IFO注文 設定後 手動介入禁止

---

## ATRの使い方

- TP/SL計算には**不使用**
- エントリー可否フィルター**専用**
- ATR(5) 5分足 基本

---

## 判定ロジック

| ATR% | 判定 | 色 |
|------|------|----|
| < thresh_ok | ✅ エントリー | 青 |
| < thresh_ng | ⚠️ ロット半減 | 黄 |
| ≥ thresh_ng | ❌ 見送り | 赤 |

### 閾値（時間足別）

| 時間足 | thresh_ok | thresh_ng |
|--------|-----------|-----------|
| 1分 | 0.8% | 1.5% |
| 3分 | 1.2% | 2.0% |
| その他 | 1.5% | 2.5% |

> ⚠️ 閾値は暫定値。実銘柄観察後に調整予定。

---

## TP/SL自動計算

現在値基準 自動算出。

```
▲ロング   TP = close × 1.012  /  SL = close × 0.994
▼ショート TP = close × 0.988  /  SL = close × 1.006
```

チャート右上ラベルに表示。最終足更新のみ。

---

## インストール

1. TradingView → Pine Editor 開く
2. コード貼り付け
3. 「チャートに追加」
4. 時間足に合わせて閾値確認
