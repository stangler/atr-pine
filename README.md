# ATR Volatility Filter + ATR倍率TP/SL + 方向性判断 — PineScript v6

株デイトレ・スキャルピング向け **ATRフィルター + 動的TP/SL + モメンタム方向性判断** インジケーター。

ATR%でボラティリティによるエントリー可否を判定し、**TP/SLをATR倍率で自動計算**。さらにRSIとStochasticで**現在ロング優勢かショート優勢か**を判断して表示します。

---

## 主な特徴

- ATR%によるエントリー可否の自動判定（青/黄/赤）
- **固定％ → ATR倍率** に変更（市場のボラティリティに自動適応）
- RSI + Stochasticによる **方向性判断**（🟢 ロング優勢 / 🔴 ショート優勢 / ⚪ ニュートラル）
- リスク・リワード比率（RRR）が安定しやすい
- チャート右上に最新情報テーブルを表示（ATR値、動的TP/SL、方向性など）

---

## トレードルール（推奨設定）

- **SL**: ATR × 1.5
- **TP**: ATR × 3.0
- **想定RRR**: 約 **1 : 2**

> ※ SL倍率・TP倍率は入力で自由に調整可能です。

**方向性判断の目安**：
- 🟢 ロング優勢 → RSI > 50 かつ Stoch %K > %D のとき
- 🔴 ショート優勢 → RSI < 50 かつ Stoch %K < %D のとき
- ⚪ ニュートラル → 上記以外

---

## ATRの使い方

- **エントリー可否フィルター**として使用
- **TP/SL計算**にも使用（動的計算）
- 推奨：ATR(14)

---

## 判定ロジック

### 1. ATR%フィルター（ボラティリティ）

| ATR%          | 判定           | 色   |
|---------------|----------------|------|
| < thresh_ok   | ✅ エントリー   | 青   |
| < thresh_ng   | ⚠️ ロット半減  | 黄   |
| ≥ thresh_ng   | ❌ 見送り      | 赤   |

**閾値（時間足別）**

| 時間足 | thresh_ok | thresh_ng |
|--------|-----------|-----------|
| 1分    | 0.8%      | 1.5%      |
| 3分    | 1.2%      | 2.0%      |
| その他 | 1.5%      | 2.5%      |

### 2. 方向性判断（モメンタム）

- **ロング優勢**：RSI(14) > 50 かつ Stochastic %K > %D
- **ショート優勢**：RSI(14) < 50 かつ Stochastic %K < %D

---

## TP/SL自動計算（ATR倍率版）

現在値（close）を基準に、**ATR倍率で動的に計算**します。

- **ロング**
  - TP = close + (ATR × TP倍率)
  - SL = close - (ATR × SL倍率)

- **ショート**
  - TP = close - (ATR × TP倍率)
  - SL = close + (ATR × SL倍率)

デフォルト設定：
- SL倍率 = 1.5
- TP倍率 = 3.0

---

## 表示内容（テーブル）

チャート右上に以下の情報をリアルタイム表示：

- ATR%
- ATR判定（エントリー/ロット半減/見送り）
- 現在値 / ATR値
- ロング TP/SL
- ショート TP/SL
- RSI値
- Stoch %K / %D
- **方向性判定**（🟢 / 🔴 / ⚪）

---

## インストール

1. TradingViewのPine Editorを開く
2. コードをすべて貼り付け
3. 「チャートに追加」をクリック
4. 必要に応じて入力値を調整
   - ATR期間（デフォルト14）
   - SL倍率 / TP倍率
   - RSI期間 / Stochastic期間
