# ATR Volatility Filter + ATR倍率TP/SL + 方向性判断 + 逆張りシグナル — Pine Script v6

株デイトレ・スキャルピング向け **ATRフィルター + 動的TP/SL + モメンタム方向性判断 + 逆張りシグナル** インジケーター。

ATR%でボラティリティによるエントリー可否を判定し、**TP/SLをATR倍率で自動計算**。  
さらにRSIとStochasticで**方向性判断**と**買われ過ぎ/売られ過ぎの逆張り目安**を表示します。

---

## 主な特徴

- ATR%によるエントリー可否の自動判定（青/黄/赤）
- 固定％ → **ATR倍率** に変更（市場ボラティリティに自動適応）
- RSI(14) + Stochasticによる **方向性判断**（🟢 ロング優勢 / 🔴 ショート優勢 / ⚪ ニュートラル）
- **逆張りシグナル**追加：買われ過ぎ（RSI≥80 & Stoch≥80） / 売られ過ぎ（RSI≤20 & Stoch≤20）
- リスク・リワード比率（RRR）が安定しやすい
- チャート右上にコンパクトな情報テーブルを表示（8行）

---

## トレードルール（推奨設定）

- **SL**: ATR × 1.5
- **TP**: ATR × 3.0
- **想定RRR**: 約 **1 : 2**

> ※ SL倍率・TP倍率は入力で自由に調整可能です。

**方向性判断の目安**：
- 🟢 ロング優勢 → RSI > 50 かつ Stoch %K > %D
- 🔴 ショート優勢 → RSI < 50 かつ Stoch %K < %D
- ⚪ ニュートラル → 上記以外

**逆張りシグナルの目安**：
- 🔴 買われ過ぎ → 逆張りロングを検討（ただし方向性も確認）
- 🟢 売られ過ぎ → 逆張りショートを検討
- ⚪ 通常レンジ → 特にシグナルなし

---

## ATRの使い方

- **エントリーフィルター**として使用
- **TP/SL計算**にも使用（動的計算）
- 推奨：ATR(14)

---

## 判定ロジック

### 1. ATR%フィルター（ボラティリティ）

| ATR%          | 判定           | 色   |
|---------------|----------------|------|
| < thresh_ok   | ✅ エントリー   | 青   |
| < thresh_ng   | ⚠️ ロット半減  | 黄   |
| ≥ thresh_ng   | ❌ 見送り      | 赤   |

**閾値（時間足別）**

| 時間足 | thresh_ok | thresh_ng |
|--------|-----------|-----------|
| 1分    | 0.8%      | 1.5%      |
| 3分    | 1.2%      | 2.0%      |
| その他 | 1.5%      | 2.5%      |

### 2. 方向性判断（モメンタム）

- **ロング優勢**：RSI(14) > 50 かつ Stochastic %K > %D
- **ショート優勢**：RSI(14) < 50 かつ Stochastic %K < %D

### 3. 逆張りシグナル

- **買われ過ぎ**：RSI ≥ 80 かつ Stoch %K ≥ 80
- **売られ過ぎ**：RSI ≤ 20 かつ Stoch %K ≤ 20

---

## TP/SL自動計算（ATR倍率版）

現在値（close）を基準に、**ATR倍率で動的に計算**します。

- **ロング**：TP = close + (ATR × TP倍率) / SL = close - (ATR × SL倍率)
- **ショート**：TP = close - (ATR × TP倍率) / SL = close + (ATR × SL倍率)

デフォルト設定：
- SL倍率 = 1.5
- TP倍率 = 3.0

---

## 表示内容（テーブル）

チャート右上に以下の情報をコンパクトに表示：

- ATR% / ATR判定
- 現在値 / ATR値
- ▲ロング TP / SL
- ▼ショート TP / SL
- 方向性判定（🟢 / 🔴 / ⚪）
- 逆張りシグナル（🔴買われ過ぎ / 🟢売られ過ぎ / ⚪通常レンジ）

---

## インストール

1. TradingViewのPine Editorを開く
2. コードをすべて貼り付け
3. 「チャートに追加」をクリック
4. 必要に応じて入力値を調整
   - ATR期間（デフォルト14）
   - SL倍率 / TP倍率
   - RSI 買われ過ぎレベル / 売られ過ぎレベル（デフォルト80/20）
   - Stochastic 買われ過ぎレベル / 売られ過ぎレベル（デフォルト80/20）

---

## 制約・既知事項

- これは**フィルター＋目安表示**用のインジケーターです（自動売買は行いません）
- テーブルは最新バーでのみ更新
- 閾値は暫定値です。実トレードで観察しながら調整してください
- 逆張りシグナルは補助的な目安として使用（単独では使わず、方向性やATRフィルターと組み合わせることを推奨）
